# 🍷 Wine Cellar App

A modern, responsive web application for managing and tracking your personal wine collection. Built with performance and user experience in mind, this app allows wine enthusiasts to easily catalog their bottles, track vintages, and manage inventory.

## ✨ Features

* **Inventory Management**: Add, edit, and remove wine bottles from your digital cellar.
* **Detailed Cataloging**: Track essential details such as vintage, varietal, region, producer, and tasting notes.
* **Smart Filtering & Search**: Quickly find specific bottles in your collection by region, grape, or optimal drinking window.
* **Modern UI/UX**: A clean, accessible, and highly responsive interface.
* **Backend Integration**: Real-time database synchronization and secure data storage.

## 🛠️ Tech Stack

This project is built using modern web development tools:

* **Frontend Framework**: [React](https://reactjs.org/)
* **Build Tool**: [Vite](https://vitejs.dev/)
* **Language**: [TypeScript](https://www.typescriptlang.org/)
* **Styling**: [Tailwind CSS](https://tailwindcss.com/)
* **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
* **Backend / Database**: [Supabase](https://supabase.com/)
* **Testing**: [Vitest](https://vitest.dev/)
* **Package Management**: npm / bun
